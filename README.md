# Bank Account Management System (C++)

A console-based banking system developed using C++.  
This project allows users to manage bank accounts and perform transactions.

## Features

- Create new bank accounts
- Deposit money
- Withdraw money
- View customer details
- Transaction history
- Transaction search by date range
- Automatic date and time recording

## Technologies Used

- C++
- File Handling
- Git & GitHub

## Project Structure

## Project Structure

Bank-Account-Management
│
├── main.cpp          # Main program with menu and program flow
├── account.cpp       # Handles account creation and account management
├── transaction.cpp   # Handles deposit, withdraw, and transaction 
├── report.cpp        # Generates reports 
├── customers.txt     # Stores customer account information
├── transactions.txt  # Stores all transaction records
└── README.md         # Project documentation

## How to Run

1. Clone the repository

git clone https://github.com/Princeverma9170/Bank-Account-Management.git

2. Open the project in VS Code or any C++ IDE

3. Compile the program

g++ main.cpp -o bank

4. Run the program

./bank
